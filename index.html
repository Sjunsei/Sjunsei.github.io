Interactive Surface

About the Project

An interactive surface programmed in Javascript where the user can influence the movement of balls bouncing within its canvas.

  • Implemented an interactive surface that can be affected by user activity
  • Intergrated existing physic formula's and behavior of random generated balls when colliding outside the visible canvas
  • Sine-wave behavior is intensified by user activity towards canvas and scales additively
  • Balls upon reaching the border changes to an alternate color of a random variety
